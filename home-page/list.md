Lists are a useful way to save variables or questions you want to return to later, or to share with others.

**Create a List**

To create a list, use the *Create a List* button and give it a name. You can add further information:

- a description and sub-title
- Principal Investiagor (your name)
- a citation
- a persistent identifier
- decide whether you want it to be public or private (ot needs to be public if you want to share)

This makes your list active and any questions or variable you add will be automatically added to that list.
To swap which list is active, click on the list name in the bottom left hand (black) panel and **Activate**

**Adding variables or questions to the list**

To add variables or questions to the list, there is a (+) when you search or browse for a variable or question. 
An example is: https://discovery.closer.ac.uk/search?Query=elephant&ResultOrdering=Relevance&IncludeDeprecated=false&PageNumber=1 
This will add the item to your **active** list.

**Removing variables or questions from the list**

Navigate to the list by clicking on the list name in the bottom left hand (black) panel. Activate the list, and click on the (-) next to the item, 
it should turn into a (+). Refresh the page or navigate to another page to save (there is no save button).
